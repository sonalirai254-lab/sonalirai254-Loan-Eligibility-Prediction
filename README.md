# 💳 Loan Eligibility Prediction

A modern web application that predicts loan eligibility based on user information. The project provides a clean, responsive interface and helps users quickly estimate whether they are likely to qualify for a loan.

## 🚀 Features

- Loan eligibility prediction
- Clean and responsive UI
- Fast prediction results
- User-friendly interface
- Easy to customize and extend

## 🛠️ Tech Stack

- HTML
- CSS
- JavaScript
- React
- TypeScript
- Tailwind CSS
- Vite

## 📂 Project Structure

```
src/
public/
components/
pages/
assets/
```

## ▶️ Getting Started

```bash
git clone https://github.com/your-username/loan-eligibility-prediction.git

cd loan-eligibility-prediction

npm install

npm run dev
```
